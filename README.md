# Auto-Typewriter

## Requirement
With pip:
- columnar 
- Pillow
- requests
- numpy
- pyfiglet

With apt:
- sudo apt-get install libopenjp2-7
- sudo apt install libtiff5
- sudo apt-get install libatlas-base-dev
